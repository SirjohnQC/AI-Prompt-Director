## Prerequisite
1. **Install Python**: [Download Here](https://www.python.org/downloads/) (Make sure to check "Add Python to PATH").
2. **Install Ollama**: [Download Here](https://ollama.com/) (Required for local analysis).
   - Once installed, run `ollama pull qwen3-vl` and `ollama pull llama3.2` in your terminal.

## How to Run (Easy Way)

### Windows
1. Download this repository.
2. Double-click **`run.bat`**.
3. The app will open in your browser automatically.

### Mac / Linux
1. Download this repository.
2. Open Terminal in the folder.
3. Run `sh run.sh`.
